## Используемые технологии

- npm 8.1.2
- node 16.13.1
- vue 3.2.13
- vuex 4.0.2

## Запуск проекта

1) Установить nodejs https://nodejs.org/en/download/
2) В папке с проектом ввести консольную команду ```npm install```
3)  ```npm run serve```
4) Проект будет доступен по адресу ```http://localhost:8080```
